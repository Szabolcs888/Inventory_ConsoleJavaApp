# Inventory console Java app

This is an inventory management console application created as part of a hands-on project. __I will try to make it bilingual in the next few days (beginning of November).__ In terms of its functions, the application registers receipt and deletion of inventory, new and existing customers and purchases. For now, it is still organized into lists, but there are also plans to integrate the database and develop a simple frontend interface.
